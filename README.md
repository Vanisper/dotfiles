# Dotfiles

当前配置面向以下环境：

- terminal : Ghostty
- shell : zsh
- shell-theme : starship
- terminal-file-mgr : yazi

> [!TIP]  
> 终端中有部分场景需要显示特殊字符，安装一种 [Nerd Font](https://www.nerdfonts.com/) 并在终端启用。

## 安装

```bash
cd ~/Documents/GitHub/dotfiles
./install.sh
source ~/.zshrc
```

## 说明

- 安装脚本会在覆盖前自动备份旧文件（`*.bak.<timestamp>`）。
